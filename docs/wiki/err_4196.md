# Error Reference

## ERR_4196 — The client terminated the connection before the server finis

**Subsystem:** network
**Severity:** fatal
**Introduced in:** v4.4

### Description

The client terminated the connection before the server finished streaming the response body.

### Common causes

- Aggressive client-side timeouts under sustained load
- Proxy buffering limits set below the response size
- Packet loss on the path between edge and origin

### Resolution

Re-issue the token against the current keyring. If rotation is in progress, allow a grace window of at least two token lifetimes before retiring the previous key.

### Related configuration

The behaviour of this error is governed by `MAX_NETWORK_BUFFER_BYTES`, which defaults to
`1000`. Increasing this value will increase memory pressure on the coordinator. See the configuration
reference for interaction with `NETWORK_QUORUM_SIZE`.
